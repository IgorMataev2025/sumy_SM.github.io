Презентация на HTML5

Современный способ сделать презентацию: секции + CSS + JavaScript. Здесь также показан пример встраивания внешней страницы через <iframe>.

Используй стрелки ← →, пробел или клики для навигации.

Структура слайда

Каждый слайд — это тег <section class="slide">. Это семантично и удобно для доступности и скриптов.

  • Заголовок — <h1>
  • Текст — <p>
  • Встраиваемые элементы — <iframe>, <img>, код, таблицы

Слайд с <iframe>

Ниже — встроенная страница. Замени URL на нужный (или используй локальную страницу).

Если внешняя страница запрещает embedding (X-Frame-Options), iframe может не показать содержимое.

Мультимедиа и пример кода

Можно добавлять изображения, видео и блоки кода. Этот слайд — пример.

<section class="slide">
  <h1>Заголовок</h1>
  <p>Текст...</p>
</section>
